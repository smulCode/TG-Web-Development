Opdracht 3

We kunnen CSS-selectors onderverdelen in vijf categorieën:

  1. Eenvoudige selectors

    Selecteer elementen op basis van naam, id, klasse.

  2. Combinator-selectors

    Selecteer elementen op basis van een specifieke relatie ertussen

  3. Pseudo-class selectors

    Selecteer elementen op basis van een bepaalde status

  4. Pseudo-elements selectors

    selecteer en stijl een deel van een element

  5. Attribute selectors

    Selecteer elementen op basis van een attribuut of attribuutwaarde

descendant selectors

Zijn afstammeling(descendants) selectors van een ander element. De syntaxis is erg eenvoudig, je geeft de naam van de parent (ouder), één of meer spaties, en dan de naam van het element dat je eigenlijke doel is. Hier volgt een voorbeeld:

ul li { }

header h2 { }

footer a { }

.module div { }

#info-toggle span { }

div dl dt a { }

Neem bijvoorbeeld ul li { }. Het betekent "elk lijst item dat afstamt van een ongeordende lijst."
