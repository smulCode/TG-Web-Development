Pseudo element voorbeelden

  1. first-line

    2. Eerste regel word opgemaakt

  2. first-letter

    3. eerste letter word opgemaakt

  3. after

    4. Met after kan je content toevoegen achter een element

  4. before

    5. Met before kan je content toevoegen voor een element

  5. Selection

    6. Met selection kan je het gedeelte van een element dat door een gebruiker is geselecteerd opmaken.

  6. backdrop

    7. is een balk ter grootte van de viewport die direct onder elk element wordt weergegeven dat in de modus Volledig scherm wordt weergegeven.

  7. cue

    8. Het ::cue CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media met VTT-tracks te stylen.

  8. first-selector-button

    9. Het ::file-selector-button CSS pseudo-element vertegenwoordigt de knop van een < input > van type="file"

  9. grammar-error

    10. Het ::grammar-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat door de user-agent is gemarkeerd als grammaticaal onjuist.

  10. marker

    11. Het ::marker CSS pseudo-element selecteert het markeringsvak van een lijstitem, dat meestal een opsommingsteken of nummer bevat. Het werkt op elk element of pseudo-element dat is ingesteld om weer te geven: list-item, zoals de elementen < li > en < summary >.

  11. part()

    12. Het ::part CSS pseudo-element vertegenwoordigt elk element binnen een schaduwboom met een overeenkomend onderdeelkenmerk.

  12. placeholder

    13. Het ::placeholder CSS pseudo-element vertegenwoordigt de placeholder tekst in een < input > of < textarea > element.

  13. slotted()

    14. het ::slotted() CSS-pseudo-element vertegenwoordigt elk element dat in een sleuf in een HTML-sjabloon is geplaatst

  14. spelling-error

    15. Het ::spelling-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat door de user-agent is gemarkeerd als onjuist gespeld.

  15. target-text

    16. The ::target-text CSS pseudo-element represents the text that has been scrolled to if the browser supports scroll-to-text fragments. It allows authors to choose how to highlight that section of text.
