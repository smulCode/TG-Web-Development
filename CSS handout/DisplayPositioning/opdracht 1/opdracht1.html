Positions die er zijn:

  1. Static

    2. Het element wordt gepositioneerd volgens de normale flow van het document en is het default van de browser

  2. Relative

    3. Het element wordt gepositioneerd volgens de normale stroom van het document en wordt vervolgens ten opzichte van zichzelf verschoven op basis van de waarden top, right, bottom en left.

  3. absolute

    4. Het element wordt verwijderd uit de normale documentstroom en er wordt geen ruimte gemaakt voor het element in de paginalay-out. Het is gepositioneerd ten opzichte van zijn dichtstbijzijnde voorouder

  4. fixed

    5. Het element wordt verwijderd uit de normale documentstroom en er wordt geen ruimte gemaakt voor het element in de paginalay-out. Het wordt gepositioneerd ten opzichte van het oorspronkelijke bevattende blok dat is ingesteld door de viewport

  5. sticky

    6. Het element wordt gepositioneerd volgens de normale stroom van het document en vervolgens verschoven ten opzichte van de dichtstbijzijnde scrollende voorouder
